# Nano Banana Pro Prompt Builder

A cinematic prompt engineering tool aimed at "Nano Banana Pro" users.
Built with **React**, **Vite**, and **Vanilla CSS**.

## Features
- **Cinematic Control Panel**: Select cameras, lenses, lighting, and angles visually.
- **Multi-Angle Consistency**: Generate consistent character variations (Wide, Close-up, Side).
- **Live Preview**: See your prompt being built in real-time.
- **JSON Export**: Export the scene configuration for external tools.

## Development

This project was built manually in an environment without Node.js.
To run locally:

1.  `npm install`
2.  `npm run dev`

## Deployment

Deploy to Vercel or Netlify.
**Note**: Vite 6 requires **Node.js 18+**. Ensure your deployment settings use Node 18 or 20.
